# ISO 19650 - 建築資訊管理標準

## 什麼是ISO 19650
ISO 19650 是建築資訊管理（BIM）的國際標準系列，提供建築資產全生命週期的資訊管理框架，確保資訊在不同階段、不同單位之間的一致性與品質。

## 標準組成

| Part | 名稱 | 發布時間 |
|-----|------|---------|
| ISO 19650-1 | 概念與原則 | 2018 |
| ISO 19650-2 | 交付階段 | 2018 |
| ISO 19650-3 | 運營階段 | 2020 |
| ISO 19650-4 | 資訊交換 | 2022 |
| ISO 19650-5 | 安全管理 | 2020 |
| ISO 19650-6 | 安全與健康資訊 | 2025 |

## 核心流程（9步驟）

1. **評估與需求** - Assessment & Need
2. **招標** - Invitation to Tender
3. **投標回應** - Tender Response
4. **任命** - Appointment
5. **動員** - Mobilisation
6. **資訊產製** - Information Production
7. **資訊交換** - Information Exchange
8. **資訊模型** - Information Model
9. **結案** - Closeout

## 重要術語

| 術語 | 說明 |
|-----|------|
| AIM | 資產資訊模型 (Asset Information Model) |
| BEP | BIM執行計畫 → IPP (2026版) |
| CDE | 共同資料環境 (Common Data Environment) |
| EIR | 業主資訊需求 |
| LOD | 資訊發展程度 → LOIN (2026版) |
| PIM | 專案資訊模型 (Project Information Model) |

## 2018 vs 2026 主要差異

| 2018版本 | 2026修訂版 |
|---------|-----------||
| BIM | IM（資訊管理）|
| BEP | IPP（資訊產製計畫）|
| LOD | LOIN（資訊需求程度）|
| 分離的交付/運營流程 | 統一9步驟流程 |
| 模型導向 | 資訊導向 |

## 2026版本預計更新
- DIS已於2026年3月發布
- 預計2027年正式實施
- 重點：從BIM轉向IM（資訊管理）
- LOD改為LOIN
- 統一的9步驟流程覆蓋全生命週期

## 台灣應用
- 許多大型專案要求符合ISO 19650
- BIM執行計畫(BEP)為常見文件
- 需注意與本地法規的配合
