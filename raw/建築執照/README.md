# 建築執照 (Building Permits)

依據建築法第 28 條，建築執照分為四種類型。

## 執照類型

| 執照類型 | 簡稱 | 用途 |
|---------|------|------|
| [建造執照/](建造執照/) | 建照 | 建築物之新建、增建、改建及修建 |
| [雜項執照/](雜項執照/) | 雜照 | 雜項工作物之建築（如水塔、廣告牌）|
| [使用執照/](使用執照/) | 使照 | 建築物建造完成後之使用或變更使用 |
| [拆除執照/](拆除執照/) | 拆照 | 建築物之拆除 |

## 每種執照的子分類

每種執照包含以下四個子分類：

| 子分類 | 說明 |
|--------|------|
| **申請文件/** | 該類執照申請所需的文件清單與格式要求 |
| **申辦流程/** | 該類執照的申請流程、審查時程與注意事項 |
| **相關法規/** | 該類執照適用的法規條文與解釋函 |
| **審核標準/** | 該類執照的審核要點與常見缺失 |

## 說明

目前各執照類型的內容正在逐步建立中。部分類型的內容可能需要根據其特有規定進行客製化調整。
