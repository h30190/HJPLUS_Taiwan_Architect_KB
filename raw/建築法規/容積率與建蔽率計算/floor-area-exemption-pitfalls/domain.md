# 容積免計項目實務陷阱

> 本模組收錄「容積免計項目」在設計與施工現場常踩的實務陷阱。
> 純法條清單只會告訴你「免計的門檻數字」，但不會告訴你「圖面要怎麼畫才不會在完工時破功」——後者才是本模組的價值所在。

## 使用情境

- 規劃梯廳、安全梯間、機電空間等容積免計項目，需確認免計條件時
- 圖面標註淨尺寸、要決定是否替裝修層預留厚度時
- 容積已接近用滿、任一免計項目失效就會超量時
- 使照查驗前自主檢查、想避免完工才爆的合規風險時

## 核心觀念：圖面標稱尺寸 ≠ 完成面淨尺寸

許多容積（及其他法規）的免計／合規門檻，查的是**完成面淨尺寸**，但建築師圖面上標的常是**結構體或粗胚的標稱尺寸**。兩者之間被「裝修層」吃掉：

```
圖面標稱淨尺寸  −  兩側裝修層厚度（石材、磚、底料）  =  完成面淨尺寸（法規實際認定值）
```

只要門檻是「**大於等於**某尺寸」，圖面就**剛好畫門檻值 = 完工必定不足**。正確做法是把裝修厚度當緩衝、往上預留。

這是一個**跨法規反覆出現的模式**（見下方「同型陷阱」）。梯廳免計容積是其中典型的一例，列為本系列一張卡片。

## 陷阱卡片：梯廳免計容積（完成面淨尺寸陷阱）

| 欄位 | 內容 |
|---|---|
| **法規依據** | 建築技術規則建築設計施工編 **第 162 條第 1 項第 1 款** |
| **法規門檻** | 每層共同使用之樓梯間、昇降機間之梯廳，其**淨深度不得小於 2.0 公尺**（關鍵字是「**淨**」深度＝量完成面）。並須同時滿足：梯廳免計面積 ≤ 該層樓地板面積 **10%**；陽臺＋梯廳之和 ≤ 該層樓地板面積 **15%**（超過部分應計入） |
| **落差機制** | 圖面標稱深度 − 兩端石材等裝修層厚度 = 完成面淨深度（被裝修層吃掉而縮減，掉破 2m 即喪失免計） |
| **圖面對策** | 圖面標稱畫到 **2.1 公尺**，預留裝修層厚度（每側約 5cm，依實際裝修做法調整），確保貼完石材後**淨深度**仍 ≥ 2.0 公尺 |
| **失敗後果** | 完成面 < 2.0m → 梯廳面積須**計入容積總樓地板面積** → 若容積率已用滿則超量 → **無法通過使照查驗、拿不到使用執照** |
| **後果時點** | 完工裝修後才會顯現，事後極難補救（可能須敲除裝修甚至調整結構） |
| **連動法規／作業** | 容積總樓地板面積計算、室內裝修、使照竣工查驗 |
| **嚴重度** | 🔴 高（直接卡使照，且屬完工後才爆的後段風險） |

## 設計檢核重點

- [ ] 凡是「免計／合規查的是**完成面淨尺寸**」的項目，圖面是否已替裝修層預留厚度？
- [ ] 門檻是「≥」時，圖面標稱值是否**高於**門檻、而非剛好等於？
- [ ] 裝修做法（石材厚度、底料、留縫）確定後，是否回頭驗算完成面淨尺寸仍達標？
- [ ] 此項目失效時，容積是否會超量？（評估嚴重度，決定預留要多保守）

---

## 條文原文（第 162 條第 1 項第 1 款，節錄）

> 「每層共同使用之樓梯間、昇降機間之梯廳，其淨深度不得小於二公尺；其梯廳面積未超過該層樓地板面積百分之十部分，得不計入該層樓地板面積。但每層陽臺面積與梯廳面積之和超過該層樓地板面積之百分之十五部分者，應計入該層樓地板面積……」
