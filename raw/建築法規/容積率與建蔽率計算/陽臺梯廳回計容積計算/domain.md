# 陽臺與梯廳回計容積計算

本模組整理建築技術規則建築設計施工編第 162 條第 1 項第 1 款中，**陽臺面積 10%**、**梯廳面積 10%**、以及**陽臺加梯廳合計 15%**的逐層檢討方式。重點不是只背上限，而是避免在 10% 已經回計後，又於 15% 檢討時重複回計同一塊面積。

## 使用情境

- 製作建照或變更設計的容積檢討表時
- 每層同時有陽臺與共同使用梯廳，需判斷免計與回計面積時
- 梯廳或陽臺任一項已超過該層樓地板面積 10%，需進一步檢討合計 15%時
- 容積接近用滿，需要清楚列出各項回計原因，避免審查時被認定重複或漏計時

## 法規門檻

依建築技術規則建築設計施工編第 162 條第 1 項第 1 款：

- 每層陽臺面積未超過該層樓地板面積 **10%** 部分，得不計入該層樓地板面積。
- 每層共同使用之樓梯間、昇降機間之梯廳，其淨深度不得小於 **2 公尺**；其梯廳面積未超過該層樓地板面積 **10%** 部分，得不計入該層樓地板面積。
- 每層陽臺面積與梯廳面積之和超過該層樓地板面積 **15%** 部分，應計入該層樓地板面積。

> 梯廳若淨深度未達 2 公尺，應先判斷其是否具備免計資格；不具備免計資格的梯廳面積，不應放入本模組的「梯廳免計候選面積」中。

## 計算變數

| 符號 | 意義 |
|---|---|
| `F` | 當層樓地板面積 |
| `B` | 當層陽臺面積總和 |
| `L` | 當層符合梯廳免計基本條件的梯廳面積總和 |
| `Cap10` | `F × 10%` |
| `Cap15` | `F × 15%` |

## 建議演算法

### 1. 先檢討陽臺 10%

```
陽臺10%回計 = max(B - Cap10, 0)
陽臺10%後免計候選 = min(B, Cap10)
```

### 2. 再檢討梯廳 10%

```
梯廳10%回計 = max(L - Cap10, 0)
梯廳10%後免計候選 = min(L, Cap10)
```

### 3. 再檢討陽臺加梯廳 15%

15% 檢討時，應使用「已通過各自 10% 檢討後仍可免計的候選面積」，而不是直接使用原始 `B + L` 再加總回計一次。

```
15%檢討基準 = 陽臺10%後免計候選 + 梯廳10%後免計候選
合計15%回計 = max(15%檢討基準 - Cap15, 0)
```

### 4. 當層總回計

```
當層總回計 = 陽臺10%回計 + 梯廳10%回計 + 合計15%回計
當層最終免計 = B + L - 當層總回計
```

## 為什麼 15% 要用 10% 後免計候選

若陽臺或梯廳已在各自 10% 檢討時產生回計面積，該超過 10% 的部分已經被計入容積。接著做 15% 合計檢討時，只應檢討「仍想免計的陽臺與梯廳面積」是否又超過 15%。

錯誤做法是：

```
先回計陽臺/梯廳各自超過10%的部分
再用原始 B + L 檢討15%
```

這會讓已經回計過的面積再次進入 15% 超量判斷，造成重複回計。

正確做法是：

```
先把 B 與 L 各自限制在 Cap10 以內
再用 min(B, Cap10) + min(L, Cap10) 檢討 Cap15
```

## 小數處理

本知識點採用以下一致口徑：

1. 用未截位的原始數值完成公式運算。
2. 各輸出欄位再以小數二位表示。
3. 小數第三位以後採**四捨五入**，不採無條件捨去。

需特別注意：若先將 `Cap10` 或 `Cap15` 四捨五入後再帶入下一步，結果可能與「原始數值運算後再四捨五入」相差 0.01 平方公尺。建議用原始數值完成全部公式，最後才將各輸出欄位四捨五入至小數二位。

## 範例 A

條件：

| 項目 | 面積 |
|---|---:|
| 當層樓地板面積 `F` | 752.58 |
| 梯廳 `L` | 78.69 |
| 陽臺 `B` | 2.70 |

計算：

```
Cap10 = 752.58 × 10% = 75.258
Cap15 = 752.58 × 15% = 112.887

梯廳10%回計 = 78.69 - 75.258 = 3.432
陽臺10%回計 = 0

梯廳10%後免計候選 = 75.258
陽臺10%後免計候選 = 2.70

15%檢討基準 = 75.258 + 2.70 = 77.958
合計15%回計 = 0
```

採小數二位四捨五入：

| 項目 | 結果 |
|---|---:|
| 梯廳10%回計 | 3.43 |
| 陽臺10%回計 | 0.00 |
| 合計15%回計 | 0.00 |
| 當層總回計 | 3.43 |

## 範例 B

條件：

| 項目 | 面積 |
|---|---:|
| 當層樓地板面積 `F` | 447.48 |
| 梯廳 `L` | 40.37 |
| 陽臺 `B` | 36.43 |

計算：

```
Cap10 = 447.48 × 10% = 44.748
Cap15 = 447.48 × 15% = 67.122

梯廳10%回計 = 0
陽臺10%回計 = 0

15%檢討基準 = 40.37 + 36.43 = 76.80
合計15%回計 = 76.80 - 67.122 = 9.678
```

採小數二位四捨五入，合計 15% 回計為 **9.68**。

## 範例 C：陽臺與梯廳都超過 10%

條件：

| 項目 | 面積 |
|---|---:|
| 當層樓地板面積 `F` | 500.00 |
| 梯廳 `L` | 62.00 |
| 陽臺 `B` | 58.00 |

計算：

```
Cap10 = 500.00 × 10% = 50.00
Cap15 = 500.00 × 15% = 75.00

梯廳10%回計 = 62.00 - 50.00 = 12.00
陽臺10%回計 = 58.00 - 50.00 = 8.00

梯廳10%後免計候選 = 50.00
陽臺10%後免計候選 = 50.00

15%檢討基準 = 50.00 + 50.00 = 100.00
合計15%回計 = 100.00 - 75.00 = 25.00

當層總回計 = 12.00 + 8.00 + 25.00 = 45.00
當層最終免計 = 62.00 + 58.00 - 45.00 = 75.00
```

這個範例可看出：10% 超過部分先回計後，15% 只檢討剩下仍欲免計的候選面積，因此不會重複計算已回計的 20.00 平方公尺。

## 檢核清單

- [ ] 梯廳是否先確認淨深度達 2 公尺，具備免計資格？
- [ ] 陽臺與梯廳是否各自完成 10% 檢討？
- [ ] 15% 檢討是否使用 `min(B, Cap10) + min(L, Cap10)`？
- [ ] 是否避免把 10% 已回計面積再次放入 15% 回計？
- [ ] 小數位處理是否全案一致，且符合地方審查表格式？
