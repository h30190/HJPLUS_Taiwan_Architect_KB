# Indoor Environmental Quality

## Overview
室內環境品質（IEQ）是影響人體健康、舒適及工作效率的關鍵因素，主要包括採光、通風、噪音、空氣品質等因素。

## Key Parameters

### 1. 室內空氣品質 (IAQ)

| 污染物 | 標準值 | 單位 |
|--------|--------|------|
| CO2 | 1000 | ppm |
| CO | 9 | ppm |
| HCHO | 0.08 | ppm |
| TVOC | 0.56 | ppm |
| PM10 | 75 | μg/m³ |
| PM2.5 | 35 | μg/m³ |
| 細菌 | 500 | CFU/m³ |

### 2. 採光環境

| 參數 | 建議值 |
|-----|--------|
| 採光係數 | ≧ 1% (工作面) |
| 自然採光比例 | ≧ 75% 樓地板面積 |
| 均匀度 | ≧ 0.4 |

### 3. 通風效率

| 參數 | 要求 |
|-----|------|
| 平均風速 | ≧ 0.2 m/s (過渡季) |
| 通風率 | ≧ 30 dm³/s.person |
| 二氧化碳濃度 | < 1000 ppm |

### 4. 聲環境

| 空間類別 | 標準值 |
|---------|--------|
| 圖書館 | ≦ 40 dB(A) |
| 辦公室 | ≦ 45 dB(A) |
| 教室 | ≦ 45 dB(A) |
| 病房 | ≦ 35 dB(A) |
| 住宅臥室 | ≦ 40 dB(A) |

## Taiwan Standards

### 法規依據
- 室內空氣品質管理法
- 建築技術規則
- 綠建築評估手冊室內環境指標

### 室內空氣品質管理法（2015年實施）
- 公告場所：學校、商場、醫療場所等
- 定期檢驗維護
- 公開檢測結果

## Design Strategies

### 通風設計
- 自然通風：開窗率、位置、高度差
- 機械通風：新風系統、全熱交換器
- 稀釋通風：導入新風稀釋污染物

### 空氣品質對策
- HVAC過濾：ULPA/HEPA過濾
- 活性碳：VOCs吸附
- 光觸媒：除菌脫臭
- 植物：室內植栽

### 採光設計
- 開窗率：≧20%外牆面
- 反射建材：提高光反射率
- 光井：中庭、天井

### 隔音設計
- 隔音量：RC≧45dB
- 吸音材料：天花、牆面
- 弹性结构：振動控制
