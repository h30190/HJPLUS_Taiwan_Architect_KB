# 低碳建築標示（LEBR）

## 什麼是低碳建築標示
低碳（低蘊含碳）建築標示制度是由內政部建築研究所於2024年實施的建築蘊含碳排評估系統，評估建築物在建材生產、運輸、施工、維護、更新及拆除階段之蘊含碳排放。

## 與建築能效的差異

| 類別 | 建築能效標示 | 低碳建築標示 |
|-----|-------------|-------------|
| 評估階段 | 使用階段 | 營建階段 |
| 碳排類型 | 使用碳排 | 蘊含碳排 |
| 實施時間 | 2022年 | 2024年 |

## 評估重點
建築全生命週期碳排放：
- **使用碳排**：營運階段（用電、用燃料）
- **蘊含碳排**：建材生產、運輸、施工、拆除

## 能效等級（7級）

| 能效等級 | 碳排減碳率 |
|---------|-----------|
| 第1級 | 最佳 |
| 第4級 | 低碳建築門檻 |
| 第7級 | 最差 |

### 低碳分級
| 低碳等級 | 條件 |
|---------|------|
| 低碳第4級 | 第4級 |
| 低碳第3級 | 第3級 |
| 低碳第2級 | 第2級 |
| 低碳第1級 | 第1級 |
| 低碳第1+級（超低碳）| 第1級+碳排減碳率≧20% |

## 蘊含碳計算方式（依EN 15978）
```
蘊含碳 = Σ(材料用量 × 碳足跡因子) + 運輸碳排 + 施工碳排
```

### 常見材料碳足跡
| 材料 | 碳足跡 |
|-----|--------|
| 鋼筋 | 1.4 kgCO2e/kg |
| 混凝土 | 0.15 kgCO2e/kg |
| 鋁門窗 | 8.5 kgCO2e/kg |
| 鋼結構 | 2.5 kgCO2e/kg |

## 減碳策略

### 材料選擇
- 使用再生材料
- 使用低碳混凝土
- 選擇在地材料

### 設計策略
- 結構系統優化
- 標準化設計
- 簡化幾何形狀

### 營造工法
- 預鑄工法
- 乾式施工
