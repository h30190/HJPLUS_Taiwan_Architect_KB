# HJPLUS 台灣建築師技能知識庫

這個儲存庫是為在台灣工作的建築師提供的綜合知識庫，包含最佳實務、設計標準、施工指引和專業技能。

## 目的

此知識庫專為幫助台灣建築師而設計，提供：
- 官方建築法規和制度資訊
- 設計和施工的最佳實務
- 專業發展資源
- 實用工具和方法論
- 結合台灣建築環境的專業知識

## 核心分類

1. **建築法規制度**
   - 台灣建築技術規則
   - 施工標準
   - 消防安全規範
   - 無障礙設計要求
   - 環境保護規範

2. **設計規劃**
   - 建築設計原理
   - 城市規劃
   - 現場分析
   - 概念設計
   - 草圖設計
   - 設計發展

3. **技術知識**
   - 結構系统
   - 機電給排水（MEP）
   - 施工材料
   - 建築技術
   - 可持續設計
   - 能源效率

4. **專案管理**
   - 專案規劃
   - 預算管理
   - 時程管理
   - 客戶關係
   - 團隊協調
   - 品質管制

5. **專業實務**
   - 契約法
   - 專業倫理
   - 保險與責任
   - 商業發展
   - 繼續教育
   - 專業執照

6. **施工材料**
   - 施工方法
   - 材料選擇
   - 品質管制
   - 安全標準
   - 環境考量
   - 維護與修繕

7. **專門領域**
   - 試圖與修復
   - 歷史建築保存
   - 醫療建築
   - 教育設施
   - 居住空間設計
   - 商業建築
   - 公共建築

## 使用方式

- 瀏覽分類尋找相關資訊
- 在分類內搜尋特定主題
- 貢獻您自己的知識和經驗
- 將知識庫作為專業實務的參考
