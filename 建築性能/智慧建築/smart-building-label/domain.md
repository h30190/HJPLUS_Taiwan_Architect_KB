# 智慧建築標章

## 什麼是智慧建築標章
智慧建築標章是由內政部建築研究所於2004年建立的建築智慧化評估制度，評估建築物透過資通訊系統整合，實現安全、健康、舒適、節能目標的能力。

## 評估指標（6大指標）

| 指標 | 說明 | 權重 |
|---------|------|------|
| 綜合佈線 | 基礎設施佈線 | 15% |
| 資訊通信 | 網路與通信系統 | 15% |
| 系統整合 | 各系統整合管理 | 15% |
| 設施管理 | 機電設施管理 | 15% |
| 安全防災 | 安防與消防系統 | 15% |
| 節能管理 | 能源監控系統 | 10% |
| 健康舒適 | 室內環境控制 | 10% |
| 智慧創新 | 創新應用 | 5% |

## 等級

| 等級 | 得分門檻 |
|-----|---------|
| 合格級 | 通過基本規定（29項）|
| 銅級 | 50-89分 |
| 銀級 | 90-119分 |
| 黃金級 | 120-139分 |
| 鑽石級 | 140分以上 |

## 基本規定（29項）示例
- 寬頻網路進線
- 住戶配缐箱
- 緊急廣播系統
- 門禁管制系統
- 中央監控系統
- 智慧電表
- 景觀照明控制
- 等

## 鼓勵項目範例
- 能源管理系統（EMS）
- 空調變頻控制
- 照明分區控制
- 空氣品質監測
- AI節能優化
- 物聯網應用

## 評估手冊版本
- 智慧建築評估手冊2024年版
- 實施日期：2024年12月1日

## 申請流程
1. 規劃設計階段 → 申請候選智慧建築證書
2. 施工階段 → 依據評估手冊施工
3. 完工取得使用執照 → 向台灣建築中心申請評定
4. 取得評定書 → 向內政部申請認可

## 申請時間
- 候選智慧建築證書：22個工作天內評定
- 智慧建築標章：50個工作天內評定
- 有效期限：5年
