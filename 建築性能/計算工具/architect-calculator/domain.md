# Architect Calculator Domain Knowledge

## 定義
建築師計算工具是整合設計過程中常用計算、法規檢查與成本估算的實用工具集。

## 核心概念

### 基本計算
- **面積計算**: 建蔽率、容積率
- **容積檢討**: 可用容積 vs 設計容積
- **結構初估**: 構件尺寸
- **機電負載**: 空調、電力
- **成本估算**: 單價 x 數量

## 台灣特別注意事項

### 1. 建蔽率與容積率

#### 定義
```
建蔽率 = (建築投影面積 / 基地面積) × 100%
容積率 = (總樓地板面積 / 基地面積) × 100%
```

#### 一般住宅區標準（參考）
```
第一種住宅區 (TH1):
- 建蔽率：50%
- 容積率：225%

第二種住宅區 (TH2):
- 建蔽率：60%
- 容積率：262.5%

第三種住宅區 (TH3):
- 建蔽率：60%
- 容積率：390%

第四種住宅區 (TH4):
- 建蔽率：60%
- 容積率：547.5%
```

#### 計算範例
```
基地面積：100 m²
建蔽率 60% → 建築投影面積 max = 60 m²
容積率 300% → 總樓地板面積 max = 300 m²

若建 4 層:
每層最大面積 = 300 / 4 = 75 m²
但受限建蔽率 = 60 m²
→ 每層實際可建 60 m²
→ 可建 5 層 (300 / 60)
```

### 2. 土地使用分區
**都市計劃法規定**

- **住宅區**: TH1-TH4
- **商業區**: C1-C5
- **工業區**: LIF (輕工業), GIF (一般工業)
- **公園用地**: 不得興建
- **道路用地**: 需退讓

### 3. 法規退縮
```
側邊無窗退縮: 1.5m
側邊有窗退縮: 2.0m
建築線退縮: 依地方自治條例
防火間距: 依建築高度與耐火時效
```

### 4. 結構初估（台灣 RC 建築）

#### 柱斷面估算
```
低層建築（1-5F）:
柱斷面 ≈ 樓面載重面積 × 0.02-0.03

例: 每柱承重 100 m²
柱斷面 = √100 × 0.025 = 35-40 cm
建議使用: 40×40 cm

高層建築:
每增加 5 層，柱斷面增加 10cm
```

#### 樑尺寸估算
```
梁高 = 跨徑 / 10 ~ / 12
梁寬 = 梁高 / 2 ~ / 3

例: 6m 跨徑
梁高 = 6000/12 = 500mm
梁寬 = 250-300mm
建議使用: 30×50 cm
```

#### 樓板厚度估算
```
住宅: 12-15 cm
辦公: 15-18 cm
停車: 18-20 cm
水池底板: 依水壓計算
```

### 7. 快速計算表

#### 建築面積試算
```javascript
function maxFloors(siteArea, coverage, far) {
  const maxBuildingArea = siteArea * coverage;
  const maxTotalArea = siteArea * far;
  return Math.floor(maxTotalArea / maxBuildingArea);
}

// 例：基地 200 m²，建蔽 60%，容積 300%
maxFloors(200, 0.6, 3.0) = 5 層
```

#### 容積檢討
```javascript
function remaining FAR(totalFAR, designFAR) {
  return totalFAR - designFAR;
}

// 可用容積 = 基地面積 × 法定容積率
// 設計容積 = ∑(各層面積)
// 剩餘容積 = 可用 - 設計
```

### 8. 常見錯誤
- **忘記扣除樓梯**: 容積計算需納入
- **陽台容積寬減**: 6m² 以內不計容積
- **機車位不計容積**: 法定停車位
- **地下室容積**: 需視用途而定
- **高層容積獎勵**: 需申請
