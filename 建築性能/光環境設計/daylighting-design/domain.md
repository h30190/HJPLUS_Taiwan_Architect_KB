# Daylighting Design Domain Knowledge

## 定義
採光設計是利用自然光滿足室內照明需求，並創造舒適光環境的設計策略。

## 核心概念

### 採光指標
- **採光係數 (Daylight Factor, DF)**: 室內照度 / 室外照度
- **日採光面積 (sDA)**: 滿足採光標準的面積比例
- **過度曝曬面積 (ASE)**: 採光過量導致不適的面積
- **統一採光指數 (UDI)**: 綜合採光評估指標

### 光環境品質
- **照度 (Lux)**: 工作面光線強度
- **均勻度**: 光線分佈均勻程度
- **眩光 (Glare)**: 過強光線導致不適
- **色溫 (CCT)**: 光線顏色溫度
- **演色性 (CRI)**: 光源顯色能力

### 設計策略
- **窗牆比 (WWR)**: 窗戶面積 / 外牆面積
- **高光窗 (Clerestory)**: 高處進光
- **採光井**: 深進深建築採光
- **反光板**: 導光板、反光遮陽

## 台灣特別注意事項

### 1. 氣候特性
**台灣日照強、濕度高，採光設計需兼顧遮陽**

- **日照時數**: 年 1,500-2,200 小時
- **太陽高度角**: 
  - 夏至正午：約 90°（直射）
  - 冬至正午：約 40°（斜射）
- **強紫外線**: 需考慮防褪色
- **高濕度**: 凝露與霉蝕風險

### 2. 法規要求
**建築技術規則建築設計施工編**

#### 採光規定（第 43 條）
```
居室採光:
- 每居室至少有 1/6 窗牆比
- 窗戶淨面積 ≥ 居室樓地板面積 1/7
- 窗戶高度 ≥ 1.5m
- 窗上沿高度 ≥ 2.0m
```

#### 通風採光（第 109 條）
```
辦公室採光:
- 窗戶面積 ≥ 樓地板面積 1/7
- 進深不宜超過 12m
```

### 3. 設計策略

#### 窗戶設計
- **窗牆比建議**:
  - 朝北：40-50%（穩定採光）
  - 朝東：30-40%（晨光適中）
  - 朝西：20-30%（西曬困擾）
  - 朝南：30-40%（需遮陽）

- **窗戶類型**:
  - 大片固定窗：採光最佳
  - 上翻窗 / 平開窗：通風採光兼顧
  - 高側窗：採光無眩光

#### 遮陽設計
- **水平遮陽**: 擋高角度太陽（南向最優）
- **垂直遮陽**: 擋低角度太陽（東西向）
- **固定遮陽**: 混凝土遮陽板
- **活動遮陽**: 百葉、簾幕

**台灣最佳遮陽設計**:
```
南向:
- 夏至太陽高度角 ≈ 90°
- 水平遮陽板深度 = 窗高 × 0.3-0.5

東西向:
- 太陽高度角低
- 需垂直遮陽或深凹窗
```

#### 玻璃選擇
- **Low-E 玻璃**: 降低熱傳 + 高透光
- **雙層玻璃**: 隔熱 + 隔聲
- **調光玻璃**: 智能控制透光率
- **彩玻璃 / 夾絲**: 裝飾性採光

### 4. 照度標準（CNS 規範）
```
住宅:
- 客廳：150-300 Lux
- 臥室：100-200 Lux
- 廚衛：200-300 Lux

辦公:
- 一般辦公室：300-500 Lux
- 會議室：300-500 Lux
- 電腦作業：500-750 Lux

商業:
- 商場：500-1000 Lux
- 展示區：750-1500 Lux
```

### 5. 眩光控制
- **直射眩光**: 太陽直射入眼
  - 對策：外遮陽、濾光玻璃
  
- **反射眩光**: 光線反射至螢幕
  - 對策：調整窗戶位置、防反光塗層
  
- **不適度 (DGP)**: 動態眩光指標
  - 舒適：< 0.40
  - 可忍受：< 0.35

### 6. 模擬工具
- **Radiance**: 專業採光模擬
- **Daysim**: 全年動態採光
- **Velux VVP**: 天窗採光計算
- **SketchUp + Plugins**: 初步模擬

### 7. 常見問題
- **西曬**: 西向房間夏季過熱
- **反光問題**: 玻璃反射影響鄰居
- **隱私**: 採光與隱私平衡
- **維護**: 高處窗戶清潔
- **熱損失**: 冬季採光 vs 保溫

## 設計流程

1. **基地分析**
   - 太陽路徑圖
   - 周圍遮擋物
   - 基地方位

2. **初步設計**
   - 窗牆比設定
   - 開窗位置
   - 遮陽概念

3. **模擬分析**
   - 採光係數計算
   - 全年採光模擬
   - 眩光評估

4. **優化調整**
   - 窗戶尺寸
   - 遮陽設計
   - 玻璃選擇

## 案例參考
- **高進深建築**: 採光井 + 反光板
- **高層建築**: 陽台遮陽 + Low-E 玻璃
- **舊建築**: 採光罩 + 管狀採光
