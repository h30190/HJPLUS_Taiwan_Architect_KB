# 綠建築標章（EEWH）

## 什麼是綠建築標章
綠建築標章是內政部建築研究所於1999年建立的綠建築評估系統（EEWH, Ecology節能、Energy節省、Waste減廢、Health健康），是全球第4個實施的綠建築評估系統，專為亞熱帶氣候設計。

## 評估指標（9項指標）

| 指標代碼 | 指標名稱 | 權重 |
|---------|---------|------|
| EA | 生物多樣性 | 6% |
| GB | 綠化量 | 6% |
| BS | 基地保水 | 6% |
| EE | 日常節能 | 21% |
| CO2 | CO2減量 | 6% |
| WR | 廢棄物減量 | 6% |
| IEQ | 室內環境 | 21% |
| RW | 水資源 | 6% |
| SC | 污水垃圾改善 | 6% |

## 適用建築類型

| 類型 | 適用對象 |
|-----|---------|
| 基本型 | 一般建築物 |
| 住宿類 | 住宅、宿舍、旅館 |
| 廠房類 | 工廠、倉庫 |
| 舊建築改善類 | 既有建築改善 |
| 社區類 | 住宅社區 |
| 既有建築類 | 已使用之建築物 |

## 認證等級

| 等級 | 分數門檻 |
|-----|---------|
| 合格級 | 通過基本指標 |
| 銅級 | 13-19分 |
| 銀級 | 20-25分 |
| 黃金級 | 26-31分 |
| 鑽石級 | 32分以上 |

## 申請流程

1. **取得建造執照** → 申請候選綠建築證書
2. **施工階段** → 依據綠建築評估手冊施工
3. **取得使用執照** → 向台灣建築中心申請評定
4. **取得評定書** → 向內政部申請認可
5. **審核通過** → 取得綠建築標章

## 一般申請時間
- 候選綠建築證書：22個工作天內評定
- 綠建築標章：50個工作天內評定
- 有效期限：5年

## 常見問題

### Q1: 一定要申請嗎？
公有建築、新建私立學校等依法令規定需申請，一般建築為鼓勵性質。

### Q2: 可以只申請部分指標嗎？
不行，綠建築標章需通過全部9項指標評估。

### Q3: 如果沒通過怎麼辦？
可進行改善後再次申請評定。
