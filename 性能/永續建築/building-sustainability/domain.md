# Building Sustainability Domain Knowledge

## 定義
綠建築與永續設計是整合環境保護、資源效率、健康舒適的建築設計方法。

## 核心概念

### 評估體系
- **LEED**: Leadership in Energy and Environmental Design
- **BREEAM**: Building Research Establishment Environmental Assessment Method
- **WELL**: 健康建築標準
- **EEARD**: 台灣綠建築評估體系

### 永續策略
- **節能降低負載**
- **水資源管理**
- **材料選擇**
- **室內環境品質**
- **基地生態**

## 台灣特別注意事項

### 1. 台灣綠建築評估體系 (EEARD)

#### 7 大指標 20 大項目

**1. 生態指標 (ECO)**
- 生物多樣性
- 基地潛能
- 綠化量

**2. 節能指標 (ENE)**
- 室內採光
- 通風
- 建築節能
- 綠能設備

**3. 省水指標 (WAT)**
- 節水設備
- 節水設計
- 中水系統

**4.  recycledMaterials (MAT)**
- 再生建材
- 耐久性

**5. 室內環境品質 (HEA)**
- 光污染
- 噪音
- 空氣品質

**6. 智慧管理 (SMG)**
- 智慧建築

**7. 施工管理 (CON)**
- 施工污染
- 施工安全

#### 認證等級
```
銀級：60 分
金級：80 分
鑽石級：100 分
```

### 2. 法規要求

#### 綠建築推動
- **政府建築**: 需申請綠建築標章
- **私立學校**: 應申請綠建築評估
- **新開發案**: 鼓勵申請

#### 建築能效證書 (EEC)
- 1,200 m²以上公共建築
- 需定期檢測申報
- 能效等級 A-F

### 3. 台灣氣候策略

#### 節能設計
- **外殼保溫**: 降低空調負載
- **自然通風**: 過渡季利用自然風
- **外遮陽**: 減少太陽熱輻射
- **綠化降溫**: 屋頂 / 牆面綠化

#### 水資源管理
- **雨水回收**: 沖馬桶 / 澆灌
- **中水回收**: 冷水機冷凝水
- **節水器械**: 省水標章
- **透水鋪面**: 減少逕流

### 4. 能源基準值

#### 台灣建築用能量
```
住宅：70-120 kWh/m²/year
辦公：150-250 kWh/m²/year
商場：250-400 kWh/m²/year
飯店：180-300 kWh/m²/year
醫院：300-500 kWh/m²/year
```

#### 淨零建築目標
- **2025**: 政府建築先行
- **2030**: 大型建築推動
- **2050**: 全面淨零

### 5. 常用策略

#### 能源
- **高效 HVAC**: VRF / 熱泵
- **LED 照明**: 省電 50%+
- **智慧控制**: BEMS 系統
- **光電整合**: 屋頂 / 外牆 PV

#### 水資源
- **省水器具**: 馬桶 / 水龍頭
- **雨水回收**: 集水 / 過濾 / 儲存
- **景觀灌溉**: 滴灌 / 霧灌

#### 材料
- **在地材料**: 減少運輸碳足跡
- **耐久性材料**: 延長使用壽命
- **低 VOC**: 室內空氣品質
- **再生材料**: 回收混凝土 / 鋼材

#### 綠化
- **屋頂綠化**: 隔熱 + 生物多樣性
- **垂直綠化**: 牆面爬藤
- **基地植栽**: 本土植物

### 6. 評估工具

#### 模擬軟體
- **EnergyPlus**: 能源模擬
- **eQuest**: 建築能耗
- **Ecotect**: 整合分析
- **PHPP**: 被動屋標準

#### 評估項目
```
年度能耗模擬:
- 冷房能耗
- 暖房能耗
- 照明能耗
- 設備能耗

碳排放計算:
- 建材碳足跡
- 施工碳排放
- 營運碳排放
```

### 7. 常見問題
- **成本高**: 初期投資增加 5-15%
- **工期長**: 綠材訂製需時間
- **維護難**: 綠化 / 光電需維護
- **認證繁瑣**: 文件眾多

### 8. 投資回報
```
初期成本增加：5-15%
營運成本降低：20-40%
投資回收年限：5-10 年
建築價值提升：5-10%
```

## 設計流程

1. **基地評估**
   - 氣候分析
   - 環境條件
   - 法規要求

2. **目標設定**
   - 綠建築等級
   - 節能目標
   - 水資源目標

3. **設計整合**
   - 建築形貌
   - 系統選定
   - 材料指定

4. **模擬驗證**
   - 能源模擬
   - 採光模擬
   - 經濟性分析

5. **審查認證**
   - 設計階段審查
   - 施工階段審查
   - 營運階段認證
