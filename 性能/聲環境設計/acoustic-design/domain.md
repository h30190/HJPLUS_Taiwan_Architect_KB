# Acoustic Design Domain Knowledge

## 定義
建築聲學設計是通過空間形狀、材料選擇與構造設計，創造舒適聲環境並控制噪音干擾。

## 核心概念

### 室內聲學
- **混響時間 (RT60)**: 聲能衰減 60dB 所需時間
- **早期衰減時間 (EDT)**: 早期聲能衰減
- **清晰度 (C50/C80)**: 語言 / 音樂清晰度
- **語言傳遞指數 (STI)**: 語言可懂度

### 隔音性能
- **STC (Sound Transmission Class)**: 空氣音隔音等級
- **IIC (Impact Insulation Class)**: 衝擊音隔絕等級
- **OITC**: 室外隔音評估

### 噪音來源
- **外部噪音**: 交通、飛機、工業
- **內部噪音**: 機電設備、人員活動
- **輻射噪音**: 管線、結構傳聲

## 台灣特別注意事項

### 1. 法規標準
**噪音管制法**

#### 噪音標準（分貝 dB）
```
第一類區域 (住宅區):
- 日間 (06:00-22:00): 55 dB
- 夜間 (22:00-06:00): 45 dB

第二類區域 (商業區):
- 日間：60 dB
- 夜間：50 dB

第三類區域 (工業區):
- 日間：65 dB
- 夜間：55 dB
```

#### 特殊區域
```
醫院、學校周邊:
- 嚴格標準 -5 dB

捷運路線:
- 依距離分區 (25m/50m/100m)
```

### 2. 建築技術規則
**建築物隔音規定**

#### 住宅隔音標準
```
牆體隔音 (STC):
- 戶與戶之間：≥ 45
- 戶與樓梯間：≥ 40

樓地板衝擊音 (IIC):
- 樓上樓下：≥ 50
```

#### 公共建築
```
餐廳鄰座:
- 桌與桌之間：STC ≥ 35

會議室:
- 對外部隔間：STC ≥ 45
```

### 3. 台灣常見噪音問題

#### 外部噪音
- **交通噪音**: 高速公路、快速道路
- **捷運噪音**: 高架段、車站
- **飛機噪音**: 松山、高雄機場
- **鐵路噪音**: 台鐵、高鐵

#### 內部噪音
- **電梯噪音**: 機房、軌道
- **空調噪音**: 主機、風管
- **管線噪音**: 給排水管
- **鄰居噪音**: 衝擊音、低頻音

### 4. 設計策略

#### 基地規劃
- **緩衝空間**: 噪音源側設置非敏感空間
- **綠化緩衝**: 樹木可降低 3-5 dB
- **聲屏障**: 擋音牆效果佳
- **距離衰減**: 增加距離降低噪音

#### 建築設計
- **窗戶隔音**:
  - 單層玻璃：STC 25-30
  - 雙層玻璃：STC 35-40
  - 夾層玻璃：STC 40-45
  
- **牆體隔音**:
  - 12cm 磚牆：STC 40-45
  - 雙層石膏板 + 吸音棉：STC 50-55
  
- **樓地板衝擊音**:
  - 地毯：IIC +10-15
  - 軟木地板：IIC +10-12
  - 懸吊地板：IIC +20-25

#### 室內聲學
- **混響時間目標**:
  ```
  講堂 (100-200 m³): 0.6-0.8 秒
  會議室 (50-100 m³): 0.5-0.7 秒
  辦公室 (開放式): 0.4-0.6 秒
  餐廳: 0.6-1.0 秒
  ```

- **吸音材料**:
  - 吸音天花板：NRC 0.7-0.9
  - 吸音板：NRC 0.8-1.0
  - 地毯：NRC 0.3-0.5

### 5. 特殊空間聲學

#### 會議室 / 講堂
- **混響時間**: 控制 0.6-0.8 秒
- **語言清晰度**: C50 > 0 dB
- **背景噪音**: NC-30 ~ NC-35

#### 餐廳
- **鄰座隱私**: 桌間隔音
- **吸音天花板**: 降低雜音
- **背景音乐**: 掩蔽對話

#### 住宅
- **臥室安靜**: 夜間 < 30 dB
- **浴室隔音**: 門 + 牆體
- **管線包覆**: 給排水管隔音棉

### 6. 模擬工具
- **ODEON**: 室內聲學模擬
- **CATT-Acoustic**: 專業音質分析
- **SoundPlan**: 外部噪音模擬
- **Insulation Calculator**: 手算工具

### 7. 常見問題
- **低頻噪音**: 難以隔離（空調、電梯）
- **側向傳聲**: 隔間牆旁繞射
- **管線共鳴**: 金屬管傳聲
- **門縫漏音**: 門縫 > 隔音失效
- **通風孔漏音**: 需加消音器

## 計算公式

### 混響時間 (Sabine 公式)
```
RT60 = 0.161 × V / A

其中:
V = 體積 (m³)
A = 總吸音量 = Σ(Sᵢ × αᵢ)
Sᵢ = 各表面面積
αᵢ = 吸音係數 (0-1)
```

### 隔音量疊加
```
R_total = 10 × log10(Σ(10^(Rᵢ/10)))
```
