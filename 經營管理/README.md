# 專業實務 (Professional Practice)

此分類涵蓋建築師專業執業的各項法律、倫理與商業實務。

## 子分類（C 類台灣法規）

| 子分類 | 關鍵主題 |
|--------|------|
| [契約法/](契約法/) | 委任契約、設計契約、服務範圍、酬金計算 |
| [專業倫理/](專業倫理/) | 公會道德規範、利益衝突、公正性 |
| [保險與責任/](保險與責任/) | 執業責任保險、專業賠償責任 |
| [專業行銷/](專業行銷/) | 作品集、參賽競賽、社群媒體、品牌曝光 |
| [專業服務費/](專業服務費/) | 酬金標準表、分割委託計法、技術服務費 |
| [專業財務管理/](專業財務管理/) | 營收預測、成本結構、財務架構 |
| [專業執照/](專業執照/) | 執業登記、開業登記、證照管理 |
| [商業發展/](商業發展/) | 業務開發、客戶獲取、策略合作 |
| [事務所營運/](事務所營運/) | 公司設立、人事管理、稅務財務 |
| [業務承接/](業務承接/) | 投圖策略、報價模式、客戶開發 |
| [創業指南/](創業指南/) | 建築師創業備忘錄、微型創業實務 |
| [繼續教育/](繼續教育/) | 在職進修（建築師法第10條）、培訓課程 |
| [爭議調處/](爭議調處/) | 公會爭議調處、訴訟與調解程序 |
| [跨國執業/](跨國執業/) | 資格互認、海外事務所、國際接案 |
