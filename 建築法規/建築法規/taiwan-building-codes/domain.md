## 第一章 基地與設計原則

### 建築線與退縮

#### 建築線（第 1 條）
```
建築物應依建築線縮建造
未經指定建築線者，不得在道路境界線建造
```

#### 側邊退縮（第 13 條）
```
無窗退縮: 1.5m
有窗退縮: 2.0m
防火牆: 可縮退 0m
```

#### 前庭後院
```
前庭: 依建築線退縮規定
後院: 無窗 0.9m，有窗 1.5m
```

## 第二章 容積設計

### 容積率計算（第 161 條）

#### 定義
```
容積率 = 總樓地板面積 / 基地面積

總樓地板面積包括:
- 各層樓地板面積
- 屋頂突出物（部分）
- 地下層（部分）

不計容積:
- 法定停車位（第 162 條）
- 設備間、機械停車（第 181 條）
- 避難層（第 300 條）
```

#### 計算範例
```
基地：200 m²
法定建蔽率：60%
法定容積率：300%

最大建築投影面積 = 200 × 60% = 120 m²
最大總樓地板面積 = 200 × 300% = 600 m²

若每層建 120 m²:
可建層數 = 600 / 120 = 5 層
```

### 建蔽率限制（第 26 條）

#### 住宅區參考值
```
第一種住宅區 (TH1): 50%
第二種住宅區 (TH2): 60%
第三種住宅區 (TH3): 60%
第四種住宅區 (TH4): 60%
```

#### 商業區參考值
```
第一種商業區 (C1): 70%
第二種商業區 (C2): 80%
第三種商業區 (C3): 80%
```

### 高度限制（第 159-1 條、268 條）

#### 法定最大高度
```
H ≦ (法定最大容積率 / 法定最大建蔽率) × 3.6 × 2

例: TH3 住宅區
容積率：390%
建蔽率：60%
最大高度 = (390/60) × 3.6 × 2 = 46.8m
```

#### 樓層高度限制（第 164-1 條）
```
住宅建築:
- 地面一層：≦ 4.2m
- 各樓層：≦ 3.6m
- 挑空部分：≦ 6m
- 夾層：≦ 6m
```

### 挑空設計（第 164-1 條）

#### 住宅挑空規定
```
位置要求:
- 每戶限 1 處
- 應於客廳或客餐上方
- 面向道路/公園/綠地（深度≧6m）

面積要求:
- 每處≧15 m²
- 合計≦總容積 1/10

高度要求:
- 挑空高度≦6m
- 旁側樓層高度合計≦6m
```

### 夾層設計（第 164-1 條）
```
設置限制:
- 仅限地面層或最上層擇 1
- 樓層高度≦6m
- 未設夾層部分需符合挑空規定
```

### 不計容積項目

#### 法定停車位（第 162 條）
```
汽車停車位: 不計容積
機車停車位: 不計容積
电梯间、樓梯: 不計容積
```

#### 陽台（第 167 條）
```
陽台面積≦6m²: 不計容積
陽台面積＞6m²: 超過部分計 1/2 容積
陽台進深: ≦1.5m
```

#### 屋頂突出物
```
機器室: ≦屋頂面積 1/10，不計容積
樓梯間: ≦屋頂面積 1/4，不計容積
水箱: 不計容積
```

## 第三章 防火避難

### 防火區劃（第 77 條）

#### 設置要求
```
樓地板面積>500m²: 需設防火區劃
每區劃面積: ≦1000m²
防火牆耐火時效: 2-4 小時
```

### 防火門（第 92 條）

#### 耐火時效要求
```
一般防火門: 1 小時
重要場所防火門: 2-3 小時
防火窗: 1-2 小時
```

### 安全梯（第 117 條）

#### 設置要求
```
3 層以上建築: 需設安全梯
11 層以上: 需設防火安全梯
31 公尺以上: 需設耐煙緩衝間
```

## 第四章 無障礙設計

### 無障礙通路（第 159 條）

#### 設置範圍
```
公共建築物: 需設無障礙設施
住宅: 10 户以上集合住宅
```

### 無障礙停車位（第 160 條）

#### 設置標準
```
停车位 10 个以下: 至少 1 個
停车位 11-50 个: 至少 2 個
停车位 51 个以上: 至少 3 個
```

## 第五章 各地特殊規定

### 台北市

#### 容積獎勵
```
都更獎勵: +25%
綠建築獎勵: +5%
開放空間獎勵: +100%（限基地面積）

#### 住宅高度限制
```
住宅區高度: ≦12m（部分區域）
高層住宅: 需經審議
```

### 台中市

#### 通風廊道
```
高層建築: 需预留通風廊

```

```

## 實務案例

### 案例 1：TH3 住宅區
```
基地：200 m²
法定建蔽率：60%
法定容積率：390%

可建條件:
- 最大建築投影：120 m²
- 最大總容積：780 m²
- 可建層數：780 / 120 = 6.5 層 → 6 層
- 最大高度：(390/60)*3.6*2 = 46.8m
```

### 案例 2：容積檢討
```
設計條件:
- 1-5F: 120 m² × 5 = 600 m²
- 6F: 100 m²
- 屋頂機械室：15 m²（不計容積）
- 陽台：8 m² × 5 = 40 m²（超過 6m² 部分計 1/2）

實際容積:
- 樓層：700 m²
- 陽台：10 m² × 6m² + (8-6)×1/2×4 = 36 m²
- 總容積：736 m²

容積率：736 / 200 = 368% < 390% ✓ 符合
```
