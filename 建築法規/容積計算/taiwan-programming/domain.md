## 容積率基本計算

### 第 161 條定義
```
容積率 = 總樓地板面積 / 基地面積

總樓地板面積計算:
- 包括：各層樓地板面積
- 包括：屋頂突出物（部分計）
- 排除：法定停車位
- 排除：陽台（部分）
- 排除：設備間（部分）
```

### 第 162 條法定停車位
```
汽車停車位：不計容積
機車停車位：不計容積
電梯前室：不計容積
樓梯間：不計容積
管理室：不計容積
```

### 第 167 條陽台
```
陽台面積≦6m²：不計容積
陽台面積＞6m²：超過部分計 1/2 容積
陽台進深≦1.5m：不計容積
陽台進深＞1.5m：超過部分計容積

例: 8m²陽台
不計：6m²
半計：(8-6)×1/2 = 1m²
總計：1m² 容積
```

### 第 181 條屋頂突出物
```
機器室：≦屋頂面積 1/10，不計容積
樓梯間：≦屋頂面積 1/4，不計容積
水箱：不計容積
機組基座：不計容積

超過限制部分：計容積
```

## 容積獎勵規定

### 綠建築獎勵

#### 獎勵基準
```
銀級：+3%
金級：+5%
鑽石級：+6%

最高：+5-6%（依地區）
```

#### 申請條件
```
需取得綠建築標章
設計階段申請
竣工階段核實
```

### 開放空間獎勵

#### 獎勵計算
```
開放面積獎勵 = 開放面積 × 獎勵倍率
獎勵倍率：100%（基地面積內）

例: 基地 200m²，設置 20m² 開放空間
獎勵容積 = 20 × 100% = 20m²
```

#### 設置要求
```
位置：基地內
通路：公眾可達
管理：公開通行維護
面積：≧10m²
```

### 都市更新獎勵

#### 獎勵項目
```
容積獎勵：+25%
容積移轉：另計
獎勵容積：可移轉

總獎勵：最高 30-40%
```

#### 申請條件
```
都更範圍內
依都更條例辦理
通過都更審查
```

### 耐震補強獎勵

#### 獎勵基準
```
耐震強化：+10-15%
耐震補強：+10%

需經耐震鑑定
需施工補強
需通過驗收
```

### 文化資產獎勵

#### 獎勵條件
```
保存歷史建築
修復傳統建築
保留文化資產

獎勵：+10-25%
```

## 容積移轉制度

### 接收基地
```
接收基地：都市計劃指定區
接收比例：依各都市計劃
移轉上限：法定容積的 30-50%
```

### 送出基地
```
史蹟紀念物
公園綠地
學校用地
道路用地
```

### 移轉計算
```
移轉容積 = 送出基地容積 × 移轉比例
總容積 = 法定容積 + 移轉容積
```

## 挑空設計規定（第 164-1 條）

### 住宅挑空

#### 位置要求
```
每住宅單位限設一處
應設於客廳或客餐廳上方
必須面向道路/公園/綠地
深度達 6m 以上法定空地
```

#### 面積要求
```
每處面積≧15m²
各處面積合計≦總容積 1/10

例: 總容積 600m²
挑空面積上限 = 600 × 1/10 = 60m²
```

#### 高度要求
```
挑空部分高度≦6m
旁側未挑空上下層高度合計≦6m

例:
挑空層高：4m
旁側層高：2×3m = 6m ✓ 符合
```

### 住宅夾層

#### 設置限制
```
僅得於地面層或最上層擇一處
設置夾層樓層高度≦6m
未設夾層部分需符合挑空規定
```

### 樓層高度限制

#### 未挑空住宅
```
地面一層：≦4.2m
其餘各層：≦3.6m
同一戶變化：最高≦4.2m
```

#### 挑空住宅
```
挑空層：≦6m
旁側層：≦3.6m
未挑空層：≦3.6m
```

## 容積檢討計算範例

### 案例 1：基本容積檢討

```
基地條件:
- 基地面積：200m²
- 法定建蔽率：60%
- 法定容積率：300%

設計條件:
- 設計層數：5 層
- 每層面積：120m²
- 陽台：8m²×5 層
- 屋頂機械室：15m²

計算:
1. 主體容積:
   5×120 = 600m²

2. 陽台容積:
   每層 8m²，6m² 不計，2m² 半計
   5×(8-6)×1/2 = 5m²

3. 機械室容積:
   屋頂面積 120m²，1/10 = 12m² 不計
   15-12 = 3m² 計容積

4. 總容積:
   600 + 5 + 3 = 608m²

5. 法定容積:
   200 × 300% = 600m²

檢討:
608m² > 600m² ❌ 超過 8m²
需減少設計面積
```

### 案例 2：獎勵容積檢討

```
基地條件:
- 基地面積：300m²
- 法定建蔽率：60%
- 法定容積率：300%

獎勵項目:
- 綠建築（金級）：+5%
- 開放空間：20m²

計算:
1. 法定容積:
   300 × 300% = 900m²

2. 綠建築獎勵:
   900 × 5% = 45m²

3. 開放空間獎勵:
   20 × 100% = 20m²

4. 總容積:
   900 + 45 + 20 = 965m²

5. 最大建築投影:
   300 × 60% = 180m²

6. 可建層數:
   965 / 180 = 5.36 層 → 5 層

檢討:
可建 5 層，每層最大 180m²
總容積 = 5×180 = 900m²
剩餘容積 = 965 - 900 = 65m²
```

### 案例 3：挑空住宅檢討

```
基地條件:
- 基地面積：200m²
- 法定建蔽率：60%
- 法定容積率：390%

設計條件:
- 每戶挑空面積：20m²
- 戶數：5 戶
- 挑空高度：5m

計算:
1. 法定容積:
   200 × 390% = 780m²

2. 挑空總面積:
   5 戶 × 20m² = 100m²

3. 挑空上限:
   780 × 1/10 = 78m²

檢討:
100m² > 78m² ❌ 超過 22m²
需減少挑空面積或增購容積
```

## 容積檢討流程

### 第一階段：法定容積計算
```
1. 確認土地使用分區
2. 查詢法定建蔽率/容積率
3. 計算法定容積
4. 計算最大建築投影
5. 計算最大高度
```

### 第二階段：獎勵容積計算
```
1. 確認可申請獎勵項目
2. 計算各項獎勵容積
3. 加總獎勵容積
4. 計算總可容積
```

### 第三階段：設計容積計算
```
1. 計算主體容積
2. 計算陽台容積
3. 計算突出物容積
4. 加總設計容積
5. 扣除不計容積項目
```

### 第四階段：容積檢討
```
1. 設計容積 vs 總容積
2. 確認容積剩餘
3. 確認建蔽率符合
4. 確認高度符合
5. 確認挑空規定符合
```

## 常見問題

### 容積超標
```
問題：設計容積 > 法定容積
解決：
- 減少樓地板面積
- 申請容積獎勵
- 購買容積移轉
- 減少層數
```

### 建蔽率超標
```
問題：建築投影 > 法定建蔽
解決：
- 縮小建築面積
- 增加退縮
- 調整配置
```

### 高度超標
```
問題：設計高度 > 法定高度
解決：
- 降低樓層高度
- 減少層數
- 申請特案
```

### 挑空超標
```
問題：挑空面積合計 > 總容積 1/10
解決：
- 減少挑空面積
- 減少挑空戶數
- 申請容積獎勵增加總容積
```
