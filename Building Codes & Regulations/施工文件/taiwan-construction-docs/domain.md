```

   - 材料檢驗

4. 施工文件
   - 廠製圖
   - 測試報告
   - 操作手冊
```

```
1. 材料要求

3. 製造與供應

   - 包裝方式
```

## 預算書編製

### 工程量計算規則

#### 混凝土工程
```
基礎：長×寬×高（m³）
柱：斷面積×高度（m³）
樑：斷面積×長度（m³）
樓版：面積×厚度（m³）
扣除：樑與柱重叠部分
```

#### 鋼筋工程
```
結構鋼筋：長度×單位重（kg）
搭接長度：40d-50d
彎摺長度：依規範計算
焊網：面積×單位重（kg）
```

#### 模板工程
```
接觸面計算（m²）
基礎側模：不含底模
柱模：四側面積
樑模：三側面積（樓版另計）
```

### 工料分析

#### RC 建築基準用量
```
混凝土柱：
- 混凝土：1 m³/m³
- 鋼筋：150-200 kg/m³
- 模板：4-5 m²/m³

混凝土樑：
- 混凝土：1 m³/m³
- 鋼筋：120-180 kg/m³
- 模板：5-6 m²/m³

混凝土樓版：
- 混凝土：1 m³/m³
- 鋼筋：80-120 kg/m³
- 模板：2 m²/m³
```

### 單價組成

#### 直接成本
```
材料費：材料原價 + 運費 + 損耗
人工費：直接工資 + 雜工
機械費：機械租金 + 燃料 + 操作
```

#### 間接成本
```
工程管理費：5-10%
利潤：5-15%
營業稅：5%
保險費：1-2%
```

### 價格調整機制

#### 營建工程物价指數
```
調整項目：
- 水泥
- 鋼材
- 砂石
- 燃料

調整頻率：每月
調整基準：完工月份指數
```

## 工程契約要項

### 中華民国工程契約範本

#### 基本條款
```
工程名稱
工程地點
造價範圍
施工期間
竣工期限
驗收標準
```

#### 付款條款
```
履約保證金：10%
預付款：無或 10%
進度款：按月計價
保留款：10%
竣工款：扣除保留後付清
保固期：1-2 年
```

#### 變更設計
```
業主變更：
- 增加：按單價計算
- 減少：按單價扣減

承包商建議：
- 需業主同意
- 價款優惠
```
