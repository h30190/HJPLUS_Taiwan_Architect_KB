# Taiwan Accessibility Domain Knowledge

## 台灣無障礙法規體系

### 法規架構
```
法律層級:
- 身心障礙者權益保障法
- 建築法

法規層級:
- 建築技術規則（無障礙設計相關條文）
- 公共建築無障礙設施規範
- 無障礙設施設計標準

自治條例:
- 各縣市無障礙環境改善自治條例
```

## 適用範圍

### 強制適用
```
新築公共建築
新築集合住宅（10 户以上）
公共運輸設施
學校醫療機構
政府辦公大樓
商業設施（面積>1000m²）
```

### 鼓勵適用
```
老舊建築改善
私有住宅
小型商業場所
```

## 無障礙入口設計

### 入口要求
```
至少一處為無障礙入口
入口處應有無障礙標誌
門寬≧0.9m
門檻高度≦2cm
自動門或推拉力≦50N
```

### 坡道設計

#### 坡度規定
```
永久坡道：≦ 1/12 (8.33%)
臨時坡道：≦ 1/10 (10%)
短坡道 (<1m): ≦ 1/8 (12.5%)
```

#### 寬度要求
```
最小寬度：1.5m
扶手淨寬：1.2m
兩側需有 10cm 空隙
```

#### 平臺規定
```
坡道起點：≧ 1.5m 深水平臺
坡道終點：≧ 1.5m 深水平臺
轉角處：≧ 1.5m×1.5m 轉彎平臺
最大垂直高度：1.5m（需設休息平臺）
```

#### 扶手設計
```
扶手高度上：0.9m
扶手高度下：0.7m
扶手直徑：3.5-4.5cm
端部延伸：≧ 30cm
兩側牆面：10cm 空隙
```

### 門檻處理
```
門檻高度：≦ 2cm
超過 2cm: 需設坡道
門檻寬度：≧ 門寬 +10cm
```

## 無障礙通路設計

### 走廊寬度
```
最小寬度：1.5m
最佳寬度：1.8m
轉角處：≧ 1.5m 清水
雙向通行：≧ 1.8m
```

### 門寬設計
```
外開門：≧ 0.9m
推拉門：≧ 0.9m
自動門：≧ 1.0m
門前等候空間：1.5m×1.5m
```

### 地面鋪面
```
防滑：濕摩擦係數≧0.6
平整：高低差≦0.5cm
警示：點字磚、黃色導引磚
無突起：避免地磚高低
```

### 樓梯設計
```
踏步步高：≦ 16.5cm
踏步步深：≧ 28cm
梯段寬：≧ 1.2m
扶手高度：0.9m
起始終點：警示地面鋪面
踏步前端：2cm 警示條
```

## 無障礙電梯設計

### 廂尺寸
```
最小尺寸：1.4m (深) × 1.1m (寬)
建議尺寸：1.6m × 1.4m
門寬：≧ 0.9m
載重：≧ 630kg（約 8 人）
```

### 按鈕設計
```
主按鈕高度：0.9-1.0m
副按鈕高度：0.7-0.8m（輪椅使用者）
按鈕大小：直徑≧2.5cm
點字標示：有
語音報站：有
燈號顯示：有
```

### 廂內設施
```
鏡子：後壁斜鏡（45°）
扶手：三側扶手（高度 0.9m）
照明：≧ 200 Lux
緊急按鈕：高度 0.9m，紅色
```

### 平層要求
```
廂地板與樓層高低差：≦ 2cm
廂門開啟時間：≧ 3 秒
廂內迴轉空間：直徑 1.5m
```

## 無障礙廁所設計

### 一般無障礙廁所

#### 尺寸要求
```
最小尺寸：1.5m × 1.5m
轉動直徑：1.5m
馬桶區：0.75m 進深
```

#### 馬桶設計
```
馬桶高度：0.45m
馬桶後方空間：≧ 0.2m
馬桶兩側空間：≧ 0.2m
```

#### 扶手設計
```
側扶手：L 型或直棒式，高度 0.7-0.8m
後扶手：直棒式，高度 0.7-0.8m
緊急拉繩：延伸至地面
```

#### 洗手檯
```
檯面高度：0.8m
膝部空間：0.75m (高) × 0.6m (寬)
水管：遮蔽或絕緣
水龍頭：槓桿式或感應式
```

#### 其他設施
```
緊急按鈕：地面 30cm，牆面 90cm
鏡子：上緣 1.0m，傾斜
置物架：高度 0.9m
門：外開或推拉，自動關閉
```

### 家庭廁所

#### 尺寸要求
```
建議尺寸：2.0m × 2.0m
容納照顧者
```

#### 加強設施
```
折疊扶手
可移動馬桶
洗澡椅
沐浴設備
```

### 嬰兒護理台
```
高度：0.8-0.9m
寬度：0.7m
長度：1.2m
安全帶：有
```

## 無障礙停車位設計

### 設置數量基準

#### 汽車停車位
```
停车位數    無障礙停车位數
1-10        1
11-50       2
51-100      3
101-150     4
151 以上     5%+5
```

#### 機車停車位
```
停车位數    無障礙停车位數
1-10        1
11 以上      2
```

### 尺寸要求

#### 汽車無障礙停車位
```
停車位寬度：3.5m
通道寬度：1.5m
總寬度：5.0m
通道位置：靠建築物側
```

#### 機車無障礙停車位
```
停車位寬度：2.0m
通道寬度：1.0m
總寬度：3.0m
```

### 標識与設施
```
地面標線：黃色
標誌牌：藍白國際符號
電極：電動車充電站（可選）
監視器：防止佔用
```

## 無障礙套房（飯店）

### 房間尺寸
```
最小尺寸：3.6m × 4.2m
迴轉空間：直徑 1.5m
```

### 門檻處理
```
門檻：無或≦2cm
各空間無高低差
```

### 浴室設計
```
無障礙馬桶
无障碍淋浴（淋浴椅）
扶手
緊急按鈕
```

### 床鋪設計
```
床高：0.5m
床側空間：≧1m
床頭櫃：扶手下方
```

## 視覺友善設施

### 警示地面鋪面
```
點字磚：入口、樓梯起始
導引磚：通路引導
顏色：黃棕對比
凸點高度：3-5mm
```

### 標示系統
```
點字標示：門牌、電梯按鈕
大字體：高度≧5cm
高對比：黑白或黃黑
語音引導：重要資訊
```

### 色彩對比
```
門檻：與地面不同色
扶手：與牆面不同色
門牌：與門面不同色
按讚：與面板不同色
```

### 照明設計
```
走廊：≧100 Lux
樓梯：≧200 Lux
電梯廂：≧200 Lux
對比區：亮度差異≦10:1
```

## 聽覺友善設施

### 視覺警報
```
閃光燈：火災警報
燈光顯示：門鈴、電話閃示
字幕顯示：電視、公告

### 輔助聽力系統
```
感應圈：售票窗、櫃台
FM 系統：會議室、禮堂
手語視訊：服務人員
```

### 通話設備
```
TTY 傳真：有
字幕電話：有
視訊手語：有
```

### 評估項目
```

標示系統
視覺友善

```

```
