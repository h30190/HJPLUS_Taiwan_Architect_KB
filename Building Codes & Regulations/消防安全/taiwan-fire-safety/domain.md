## 消防設備設置標準

### 火災警報設備

#### 設置條件
```
樓地板面積 > 100 m²: 需設手動警報
地下層、高層：需設自動火警探測
特定場所：全天候監視
```

#### 警報器配置
```
室外警報器：1 具
室內警報器：每一防火區劃至少 1 具
探測器間距：依類型而定
```

### 消防栓設備

#### 設置基準
```
樓地板面積 > 100 m²: 強制設置
每一處需有 2 條以上水管到達
服務半徑：≦ 30m
```

#### 配置要求
```
水帶長度：20-30m
消防箱尺寸：70×70×160cm
栓口位置：距地 70-100cm
```

### 火灑水系統

#### 設置條件
```
地下 1 層以下：強制設置
高層建築（31m 以上）：強制設置
樓地板面積 > 1,000 m²: 強制設置
停車空間：強制設置
```

#### 配置標準
```
灑水頭間距：≦ 4.5m
保護面積：≦ 12 m²/頭
動作温度：68°C（一般）
```

### 泡沫滅火設備

#### 適用場所
```
油庫、油槽
變壓器室
車庫
```

#### 設置要求
```
泡沫濃度：3% 或 6%
供給時間：≧ 10 分鐘
泡沫液儲存：≧ 2 次用量
```

### 乾式化學藥劑滅火設備

#### 適用場所
```
電氣室
機房
特殊設備室
```

### 自來水消防設備

#### 使用條件
```
水壓充足地區
經主管機關核准
需備有消防泵
```

## 避難逃生設備

### 防排煙設備

#### 設置條件
```
地下層：需設排煙設備
無窗辦公室：需設機械通風
樓梯間：需設加壓送風
走廊長度 > 20m: 需設排煙
```

#### 配置要求
```
排煙口面積：≧ 樓板面積 1/40
加壓送風量：≧ 5 L/s·m²
排煙量：≧ 10 L/s·m²
```

### 緊急照明與誘導燈

#### 設置位置
```
樓梯間：每層均需設置
走廊：間距≦10m
避難通道：全程照明
出口標誌：各出口處
```

#### 照明要求
```
照度：≧ 10 Lux
持續時間：≧ 1 小時
電源：儲備電源≧90 分鐘
```

## 防火區劃

### 設置條件
```
樓地板面積 > 500 m²: 需設防火區劃
地下層：需設防火區劃
混合使用建築：不同用途間需分隔
```

### 面積限制
```
設置火灑水：≤ 2,000 m²
未設火灑水：≦ 1,000 m²
地下室：≦ 500m²
停車空間：≦ 2,000 m²
```

### 防火牆

#### 構造要求
```
耐火時效：2-4 小時
不得有開口
開口處需設防火門
牆體應延伸至屋頂
```

### 防火門

#### 耐火時效
```
一般防火門：1 小時
重要場所防火門：2-3 小時
爐門、電氣室門：2-3 小時
```

#### 配置要求
```
需設自動關閉裝置
需設門閂裝置
開啟方向：順避難方向
寬度：≧ 0.9m
```

## 避難設施

### 安全梯

#### 設置條件
```
3 層以上：需設安全梯
11 層以上：需設防火安全梯
31m 以上：需設耐煙緩衝間
地下室：需設安全梯
```

#### 構造要求
```
耐火時效：2-4 小時
樓梯寬度：≧ 1.1m
樓梯高度：≦ 18 階/段
扶手高度：0.9m
```

### 避難層

#### 配置要求
```
地面一層：作為避難層
樓層高度：≦ 5m
地板載重：≧ 200 kg/m²
```

### 避難通道

#### 走廊寬度
```
最小寬度：1.2m
人流量大：≧ 1.5m
無障礙通道：≧ 1.5m
```

#### 走廊長度
```
住宅：≦ 20m
辦公：≦ 25m
商場：≦ 30m
設置火灑水：可放寬 1.2 倍
```

### 避難窗口

#### 設置條件
```
地下室：需設避難窗
窗下高度：≦ 1.2m
窗戶面積：≧ 0.5 m²
開啟方式：易於開啟
```

## 特殊場所規定

### 停車空間

#### 防火區劃
```
樓地板面積：≦ 2,000 m²
需設防火門
需設排煙設備
```

#### 消防設備
```
消防栓：間距≦30m
火灑水：強制設置
滅火器：每 4 車位 1 具
```

### 地下層

#### 加強要求
```
防火區劃：≦ 500 m²
火灑水：強制設置
排煙：強制設置
樓梯：需設防火安全梯
```

### 高層建築

#### 定義
```
31m 以上建築物
11 層以上建築物
```

#### 加強設備
```
耐煙緩衝間
加壓送風設備
緊急廣播
消防專用水泵
```
