# Building Services Domain Knowledge

## 定義
建築機電系統是維持建築機能運作的基礎設施，包括空調、給排水、電氣、弱電、消防等系統。

## 核心概念

### 空調系統 (HVAC)
- **冷負載計算**: 傳熱 + 通風 + 人員 + 設備
- **系統類型**:
  - VRF (變冷媒流量): 住宅、小型辦公
  - AHU (全空氣): 大空間、商場
  - FCU (室內機): 飯店、辦公
  - Chiller (冷水機): 高層、大型建築

### 給排水系統
- **給水**: 自來水 + 中水 + 雨水回收
- **排水**: 生活污水 + 雨水
- **熱水**: 太陽能 + 熱泵 + 燃气

### 電氣系統
- **配電**: 高壓進線 → 變壓器 → 低壓配電
- **照明**: 一般照明 + 緊急照明
- **電力**: 插座迴路
- **緊急**: 發電機 + UPS

### 弱電系統
- 網路通訊
- 監控保全
- 防火警報
- 廣播疏散

### 消防系統
- 消防栓
- 灑水頭
- 防火門
- 防排烟

## 台灣特別注意事項

### 1. 氣候考量
**台灣高溫高濕，空調為主要機電負載**

- **全年冷房**: 僅冬季短期暖房需求
- **高濕度**: 除濕負載重要
- **颱風**: 室外機固定與進風口防護
- **梅雨**: 防水與防霉考量

### 2. 空調設計

#### 冷負載估算（經驗值）
```
住宅：150-200 W/m²
辦公：180-250 W/m²
商場：250-350 W/m²
飯店客房：150-200 W/m²
餐厅: 300-450 W/m²
```

#### 通風需求（建築技術規則）
```
辦公空間：25 L/s per person
住宅：30 L/s per person
餐厅: 40 L/s per person
```

#### 常見系統
- **住宅**: VRF 分離式
- **辦公**: VRF / 水冷箱機
- **高層**: 冷水機 + FCU
- **廠房**: 工業通風 + 局部空調

### 3. 給排水設計

#### 用水基準（建築技術規則）
```
住宅：300-500 L/person/day
辦公：80-150 L/person/day
飯店：600-1000 L/person/day
商場：60-100 L/person/day
```

#### 法規要求
- **省水標章**: 水栓、馬桶、淋浴需符合 CNS
- **中水回收**: 2,000 m²以上建築需設置
- **雨水回收**: 新開發基地需考量
- **熱水供應**: 集合住宅需集中式

### 4. 電氣設計

#### 負載估算（經驗值）
```
住宅：60-100 VA/m²
辦公：120-180 VA/m²
商場：150-250 VA/m²
飯店：100-150 VA/m²
```

#### 配電系統
- 小型建築：單相 220V / 三相 380V
- 大型建築：高壓進線（11.4kV / 22.8kV）
- 緊急電源：發電機（消防 + 緊急照明）

### 5. 消防系統

#### 設置基準（各類場所消防安全設備設置標準）
```
消防栓:
- 樓地板面積 100 m²以上
- 每一處需有 2 條以上水管到達

灑水系統:
- 地下 1 層以下
- 高層建築（31 公尺以上）
- 樓地板面積 1,000 m²以上

防火門:
- 防火區劃開口
- 樓梯間出入口
```

#### 防排烟
- 無窗辦公室需機械通風
- 樓梯間需加壓送風
- 地下空間需排煙設備

### 6. 設備空間

#### 機房配置
- **主機房**: 屋頂 / 地下室
- **電氣室**: 各樓層 / 地下室
- **電話機房**: 各樓層
- **管井**: 垂直管線穿越

#### 最小尺寸（參考）
```
電氣室：3m × 3m（小型）
弱電室：2m × 2m
管井：60cm × 60cm（最小）
```

### 7. 常見問題
- **管線衝突**: 空調風管 vs 消防水管 vs 電氣桥架
- **機房空間不足**: 規劃階段未預留
- **管井尺寸不足**: 管線無法施工
- **震動噪音**: 機具隔震與隔音不足
- **冷卻水塔**: 水滴飛散與鄰居干擾

## 設計流程

### 1. 需求分析
- 建築用途與使用者
- 使用時間與負載
- 特殊需求（實驗室、餐廳）

### 2. 系統規劃
- 選擇適當系統
- 估測負載
- 配置設備位置

### 3. 管線規劃
- 水平管線（樓層）
- 垂直管線（管井）
- 管線綜合（BIM）

### 4. 法規審查
- 建築技術規則設備編
- 消防安全法規
- 電業法規
- 環保法規
