# 技能模組整合架構

## 簡介

本知識庫採用參考repo的技能模組概念，但結合台灣建築師的實際需求，將18個技能模組整合到原有7大分類架構中。

## 技能整合至七個主要分類

### 1. 建築法規制度
- 法規與合規要求技能模組
- IBC、ADA、消防法規
- 台灣建築技術規則
- 無障礙設計要求

### 2. 設計規劃
- 設計理論技能模組
- 建築設計哲學
- 前沿分析方法
- 概念設計技巧

### 3. 技術知識
- 結構系統技能模組
- 機電給排水技能模組
- 建築技術知識
- 可持續設計技能

### 4. 專案管理
- 專案規劃技能模組
- 預算管理技能
- 時程管理技能
- 客戶關係技能

### 5. 專業實務
- 契約法技能模組
- 專業倫理技能
- 商業發展技能
- 專業執照技能

### 6. 施工材料
- 施工方法技能模組
- 材料選擇技能
- 品質管制技能
- 安全標準技能

### 7. 專門領域
- 試圖與修復技能
- 歷史建築保存技能
- 醫療建築技能
- 教育設施技能
- 居住空間設計技能
- 商業建築技能
- 公共建築技能
