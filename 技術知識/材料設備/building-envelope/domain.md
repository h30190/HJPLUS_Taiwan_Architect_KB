# Building Envelope Domain Knowledge

## 定義
建築皮殼（Building Envelope）是分隔建築室內外環境的封閉系統，包括外牆、屋頂、外窗、樓板等構件。

## 核心概念

### 熱工性能
- **U-value（傳熱係數）**: 表示單位面積、單位溫差下的傳熱量
- **R-value（熱阻值）**: R = 1/U，數值越大保溫效果越佳
- **SHGC（太陽得熱係數）**: 透過玻璃進入室內的太陽輻射比例
- **熱橋效應**: 構件接頭處因熱阻減小而產生的額外熱損失

### 防水系統
- **排水式防水**: 靠外層防水 + 排水間隙設計
- **Barrier 防水**: 靠連續防水層阻擋水汽
- 台灣多雨氣候需注意排水設計

### 氣密性設計
- 影響空調能效與室內空氣品質
- 台灣建築需注意颱風負壓影響

## 台灣特別注意事項

### 1. 氣候特性
- **高溫高濕**: 年平均溫度 23°C，相對濕度 70-80%
- **梅雨季**: 5-6 月持續降雨，需加強防水
- **颱風**: 夏季颱風季需考慮負壓與飛来物
- **地震**: 構造接頭需考慮耐震變形能力

### 2. 法規要求
- **建築技術規則第 9 章**: 通風、採光、隔熱
- **CNS 3815**: 建築物能效標準
- **Green Mark 綠建築標章**: 外殼熱性能要求
- TODO: 整合 EEC 建築能效證書規範

### 3. 常用材料體系
- **外牆**:
  - RC 外牆 + 外保溫系統（ETICS）
  - 預鑄構件（PC）
  - 帷幕牆（Curtain wall）
  
- **屋頂**:
  - 綠色屋頂（Green roof）
  - 反射屋頂（Cool roof）
  - 太陽能光電整合（BIPV）
  
- **窗戶**:
  - Low-E 雙層玻璃
  - 断熱橋鋁窗框
  - 外遮陽整合

### 4. 常見問題
- **凝露**: 高濕度氣候下，冷橋處易產生凝露
- **漏水**: 梅雨季與颱風季需特別注意防水
- **褪色**: 強紫外線导致建材褪色加速
- **膨脹**: 溫差大需考慮材料熱漲冷縮

## 設計策略

### 被动式设计
1. **適宜窗牆比**: 台灣氣候 WWR 建議 30-40%
2. **外遮陽**: 水平遮陽檔太陽輻射最有效
3. **熱質量**: 利用熱延遲降低尖峰負載
4. **自然通風**: 配合夏季通風降低冷房負載

### 材料選擇
1. **高反射率**: 淺色外飾面反射太陽輻射
2. **外保溫優於內保溫**: 減少熱橋、保護結構
3. **斷熱橋設計**: 金屬構件需斷熱處理
4. **耐候性**: 抗鹽蝕（沿海）、抗紫外線

## 計算範例

```
牆體 U-value 計算:
R_total = R_si + R_1 + R_2 + ... + R_so
U = 1 / R_total

其中:
R_si = 0.12 m²·K/W (室內表面熱阻)
R_so = 0.03 m²·K/W (室外表面熱阻，有風)
```
