# Structural Systems Domain Knowledge

## 定義
結構系統是建築承受與傳遞荷重至基礎的骨架系統，包括垂直構件（柱、牆）、水平構件（樑、樓板）與基礎。

## 核心概念

### 荷重分類
- **靜荷重 (Dead Load)**: 結構自重 + 固定構材
- **活荷重 (Live Load)**: 使用上可變荷重
- **風荷重 (Wind Load)**: 颱風與季風影響
- **地震力 (Seismic Force)**: 地震時慣性力
- **其他**: 雪荷重（高山地區）、溫差應力

### 結構體系
- **耐震牆系統 (Shear Wall)**: 抗側力主要系統
- **框架系統 (Moment Frame)**: 樑柱剛接
- **撐牆系統 (Braced Frame)**: 斜撐抗側力
- **混合系統**: 兩者結合

### 結構材料
- **RC (Reinforced Concrete)**: 鋼筋混凝土
- **SC (Steel Concrete)**: 鋼骨混凝土
- **Steel**: 純鋼結構
- **Timber**: 木構造（新興 CLT）
- **Hybrid**: 混合構造

## 台灣特別注意事項

### 1. 地震風險
**台灣位於環太平洋地震帶，地震頻繁**

- **地震分區**: 台灣分為 4 級地震區
  - 第 1 級：一般地區
  - 第 2-4 級：地震危險度依次增加
  
- **地盤分類**: E1-E5 共 5 級
  - E1: 堅硬地盤
  - E5: 軟弱地盤（需特別加強）

- **設計地震係數**:
  ```
  Ca = A · Z · I / (R · S)
  
  其中:
  A = 地質調整係數 (台灣 0.06-0.32)
  Z = 地震區分係數 (1.0-1.5)
  I = 重要度係數 (1.0-1.5)
  R = 構造減弱係數
  S = 地盤係數
  ```

### 2. 颱風風壓
- **基本風速**: 台灣各地 37-63 m/s
- **沿海地區**: 需特別加強
- **高層建築**: 需考慮風振效應
- **屋頂掀除**: 固定需強化

### 3. 法規要求
**建築技術規則建築構造編**

- **第 2 章 抗震**: 
  - 最大高度限制
  - 最小構造柱配置
  - 耐震係數計算
  
- **第 3 章 構造要項**:
  - RC 構造細節
  - 鋼構造接合
  
- **最小配筋率**:
  - 構造柱：0.002-0.005
  - 樑：依彎矩計算

### 4. 常用結構體系

#### 住宅建築
- **低層（1-3F）**: RC 框架 + 磚牆
- **中層（4-10F）**: RC 框架 + 耐震牆
- **高層（11F+）**: RC 框壁構造 / SC

#### 商業建築
- **辦公大楼**: RC/SC 核心牆 + 外框架
- **廠房**: 鋼構造 + 預力樓板
- **商場**: 長跨徑桁架/拱結構

### 5. 基礎型式
- **獨立基礎**: 低層 RC 建築
- **連繫基礎**: 柱距密、荷重大
- **版基礎**: 地質軟弱、高層建築
- **樁基礎**: 深層地盤、超高層

### 6. 常見問題
- **軟弱地盤**: 板橋、永和等新店溪沖積層
- **液化現象**: 沿海與沖積平原需注意
- **沉陷差異**: 新旧建築交界、地質變化處
- **鹽蝕**: 沿海地區鋼筋腐蝕加速

## 構件尺寸初估

### RC 柱（經驗公式）
```
柱斷面邊長 ≈ √(P / (0.5 × fc'))
P = 載重 (ton)
fc' = 混凝土強度 (MPa，通常 210-350)

簡易估算:
低層: 柱斷面 ≈ 樓面載重面積 × 0.02-0.03
高層: 柱斷面相依層數增加 10-20%
```

### RC 樑
```
梁高 ≈ 跨徑 / 10 ~ / 12
梁寬 ≈ 梁高 / 2 ~ / 3

例: 6m 跨徑樑
梁高 = 6000/12 = 500mm
梁寬 = 250-300mm
```

### 樓板厚度
```
單向板: 厚度 ≈ 跨徑 / 25 ~ / 30
双向板: 厚度 ≈ 跨徑 / 30 ~ / 35
樓板最小厚度: 120mm（住宅）
```

## 設計流程

1. **結構規劃**
   - 決定結構體系
   - 配置垂直構件
   - 確保對稱性（抗震）

2. **荷重計算**
   - 靜荷重：材料重 + 隔牆
   - 活荷重：依使用類别
   - 地震力：等效側力法
   - 風荷重：外壓係數

3. **結構分析**
   - 手算初估
   - 電腦模擬（ETABS/SAP2000）

4. **細節設計**
   - 配筋計算
   - 接頭設計
   - 基礎設計
